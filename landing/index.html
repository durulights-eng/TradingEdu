ChartMon

나만의 AI 주식 차트 트레이너 (AI Stock Chart Trainer)

About ChartMon (서비스 소개)

ChartMon is a candlestick trading training and simulation application. It helps users practice technical analysis of stock and coin charts using historical data. Users can solve daily quizzes, trace patterns, analyze their trading habits, and improve their decision-making skills in a simulated environment.
(차트몬은 주식 및 코인의 과거 차트 데이터를 활용해 캔들 매매를 시뮬레이션하고, 학습 성향을 분석해주는 교육용 차트 트레이닝 모바일 애플리케이션입니다.)

Google User Data Usage (구글 로그인 정보 이용 목적)

ChartMon requests Google authentication (Sign-in) to securely identify users and synchronize their training data (such as daily streaks, XP, solved quiz logs, and competitive ratings) across different mobile devices. Google user data is used solely for authentication and profile management, and is never shared with third parties.
(구글 로그인 정보는 사용자 식별 및 학습 정보의 기기 간 동기화 목적으로만 투명하게 사용되며, 제3자에게 배포되거나 타 목적으로 일체 사용되지 않습니다.)

Core Features (핵심 기능)

  • Daily Routine (일일 맞춤 훈련): Custom generated quizzes tailored to the user's trading rating.
  • Drill Gym (테마별 훈련소): Specialized focus packs on various patterns (breakouts, supports, indicators).
  • Analytics (매매 능력 분석): Account statistics mapping user weaknesses in key trading disciplines.
  • Library (트레이더 이론 백과): Conceptual guidebooks with hand-drawn trading diagrams.
