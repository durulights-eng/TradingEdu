# ChartMon: Initial Launch & Business Strategy Plan (초기 사업 및 마케팅 실행 계획서)

본 문서는 주식 차트 트레이딩 교육 앱 **ChartMon**의 초기 런칭을 위한 법인 행정, 프로덕트 개발, 인스타그램 마케팅, 그리고 구글 플레이스토어 출시까지의 End-to-End 실행 계획을 정의합니다.

---

## 1. 프로젝트 개요 (Executive Summary)
* **서비스명**: ChartMon (차트몬)
* **슬로건**: *"하루 15분, 주식 차트 트레이딩을 듀오링고처럼 마스터하세요"*
* **핵심 가치**: 
  1. 지루한 이론 학습 대신 **실전 차트 시뮬레이션 기반의 퀴즈**를 통해 실전 감각 배양.
  2. 출석 스트릭(Streak), XP, 레벨 시스템 등 **게이미케이션(Gamification)** 요소를 적용하여 지속적인 학습 동기 부여.
  3. 모바일 환경에 최적화된 하이브리드 앱(Capacitor)을 통해 플레이스토어 신속 출시.

---

## 2. 6단계 비즈니스 로드맵 (6-Phase Roadmap)

### Phase 1: 법인/행정 및 기술 기반 구축 (완료 및 진행 중)
* **행정 인프라**: 법인 설립 완료, 사업자등록증 발급 완료, 구글 개발자 계정 확보 완료.
* **기술 스캐폴딩**: React + TypeScript + Vite + Capacitor 기반의 안드로이드 패키징 환경 셋업 완료.
* **버전 관리**: GitHub 비공개 리포지토리 생성 및 로컬 코드 동기화 예정.
* **구글 개발자 콘솔 연동**:
  * 구글 플레이 콘솔에 법인 사업자등록번호 및 대표자 정보 인증.
  * 수익 창출을 위한 법인 계좌 연동 및 판매자 프로필(Merchant Account) 설정.

### Phase 2: 기술 콘텐츠 및 퀴즈 DB 고도화
* **전문 교육 자료 확보**: [docs/trading-theories/](file:///d:/TradingEdu/docs/trading-theories/)에 정리된 6개의 핵심 카테고리(캔들스틱, 지지/저항, 추세선, 차트패턴, 보조지표, 리스크관리)를 인앱 `TheoryReader`에 내재화.
* **퀴즈 DB 확충**: 초기 MVP 출시 시점 기준 카테고리별 10개 이상, 총 60개 이상의 고품질 실전 차트 문제 확보.
* **차트 드로잉 정밀화**: Canvas 엔진을 고도화하여 실제 역사적인 차트 사례(예: 2020년 테슬라 런칭 돌파, 2022년 비트코인 붕괴 등)의 데이터 포인트를 이식.

### Phase 3: 게이미케이션 및 결제 모듈 탑재
* **출석 스트릭(Streak) 로직**: 로컬스토리지를 넘어 백엔드 서버(Supabase 등) 또는 기기 내 암호화 데이터베이스에 접속 기록 저장 및 연속 활성 시 보상 지급.
* **구글 플레이 인앱 결제 (Google Play Billing) 연동**:
  * 외부 웹 결제(Toss/Portone)를 배제하고, 구글 플레이 정책에 맞게 **구글 플레이 인앱 구독 (Google Play Billing Library)**만 단독 적용합니다.
  * Capacitor 환경에서는 `@capgo/capacitor-in-app-purchase` 또는 `cordova-plugin-purchase` 플러그인을 활용해 JavaScript 단에서 결제 요청을 처리합니다.
  * 구글 플레이 콘솔의 **'수익 창출 -> 제품 -> 정기 결제(Subscriptions)'** 메뉴에서 상품ID(예: `premium_monthly`)를 등록하고 가격(월 9,900원 예정) 및 결제 주기를 설정합니다.
* **구독 모델 설정**:
  * **무료 티어**: 매일 3회의 퀴즈 제공 및 기초 캔들스틱 이론 오픈.
  * **프리미엄 구독 (월 9,900원 예정)**: 무제한 퀴즈, 맞춤형 오답 노트 트레이닝, 심화 보조지표 분석 이론 및 VIP 텔레그램/디스코드 커뮤니티 입장권 부여.

### Phase 4: 인스타그램 비즈니스 셋업 및 콘텐츠 마케팅
인스타그램은 ChartMon의 가장 강력한 고객 획득 채널(User Acquisition Channel)입니다.

* **프로필 및 바이오(Bio) 세팅**:
  * 인스타그램 비즈니스 계정 전환 및 Facebook 법인 비즈니스 페이지 연결.
  * 링크트리(Linktree)를 프로필에 걸어 웹 랜딩페이지로 연결하거나 구글 플레이스토어 설치 링크 연결.
* **Reels 마케팅 쇼츠 콘텐츠 기획**:
  * **패턴 낚시 폭로 콘텐츠**: *"헤드앤숄더 오른쪽 어깨에서 매도 치면 90% 청산당합니다"* -> 넥라인 돌파 확인의 중요성을 15초 내외 숏폼으로 제작.
  * **비교 대조 콘텐츠**: *"차트에 낙서하는 개미 vs 진짜 돈 버는 작도법"* -> 지저분한 차트와 심플한 3선 차트를 비교.
  * **리스크 관리 자극 콘텐츠**: *"승률 90% 기법이 계좌를 깡통으로 만드는 이유"* -> 물타기로 한 번에 파산하는 과정 애니메이션 시각화.
  * **Call to Action (CTA)**: 영상 마지막 3초에 "하루 15분 차트 트레이딩을 듀오링고처럼. 프로필 링크에서 앱 다운로드!" 배치.

### Phase 5: 자존심 자극형 마이크로 퀴즈 퍼널 및 랜딩페이지 구축
광고 클릭 후 즉시 설치를 유도하지 않고, 유저의 '트레이딩 지능 자존심'을 자극하는 초간단 실력 진단 퀴즈 퍼널을 거치게 하여 이탈율을 줄이고 최종 다운로드 전환율을 극대화합니다.

> [!IMPORTANT]
> **핵심 광고 전략 (중요):**
> 실제 광고에서는 **'5단계 등급 상승'을 전면 광고하지 않습니다.** 대신, **'각 유저의 오답 패턴과 진단 결과에 따라 난이도가 자동 튜닝되는 1:1 맞춤형 차트 운동(Personalized Difficulty Customization)'**을 소구점으로 활용합니다. "초보자에게는 기초 캔들부터, 숙련자에게는 휩소 및 리스크 관리 퀴즈부터 학습되도록 최적의 학습 경로를 튜닝해 준다"는 점을 강조합니다.

* **초간단 3단계 마이크로 퍼널 (Micro-Quiz Funnel) 설계**:
  1. **1단계 (가벼운 기술 퀴즈 - 발 들이기)**: *"상위 5% 트레이더만 바로 맞추는 차트 패턴 퀴즈: 아래 그림의 돌파 신호 중 가짜 돌파(Bull Trap)는 몇 번일까요?"* (흥미 유발 및 문제 풀이 시도 유도).
  2. **2단계 (투자 자가 진단 - 매몰 비용 형성)**: *"당신의 현재 투자 경력과 차트 작도 실력은 스스로 몇 점이라고 생각하시나요?"* (사용자가 직접 짧은 시간을 투자하여 응답함으로써 퍼널 잔류율 상승).
  3. **3단계 (결과 피드백 및 자존심 자극 - 액션 유도)**: *"당신의 트레이딩 감각은 '뇌동매매 돌파 추종자' 등급입니다. 손실 확률 82%. 이 상태로 계속 매매하면 시장의 먹잇감이 됩니다. 뇌의 차트 인식 회로를 개조하십시오. 퀴즈 진단 결과 분석을 바탕으로, 당신의 눈높이에 100% 맞춤 튜닝된 차트 훈련을 바로 제공합니다. 하루 15분 ChartMon으로 맞춤형 워크아웃을 시작하세요!"* -> 다운로드(구글 플레이) 유도.
* **메타(페이스북/인스타그램) 광고 집행**:
  * **광고 소재**: *"주식/코인 경력 3년 차도 90% 틀리는 차트 IQ 테스트. 당신의 오답 패턴을 분석해 1:1 맞춤형 차트 트레이닝 문제를 생성해 드립니다."* (난이도 자동 튜닝 및 자율 훈련 강조)
  * **광고 목표**: 마이크로 퀴즈 랜딩페이지 유입 -> 앱 설치 및 구독 전환.

### Phase 6: 앱 런칭 및 리텐션 부스팅
* **그랜드 오픈**: 사전 예약 신청자 대상 이메일/SMS 알림 발송 및 인앱 첫 구독 할인 혜택 제공.
* **유저 피드백 루프**: 
  * "오늘 풀었던 퀴즈가 유익했나요?" 평가 수집.
  * 앱 평점 5점 유도 이벤트를 진행하여 구글 플레이스토어 상위 노출(ASO - App Store Optimization) 확보.
  * 리텐션 분석(D+1, D+7, D+30 재방문율)을 통해 이탈 구간의 퀴즈 난이도 조절.

## 3. Supabase 서버리스 백엔드 아키텍처 (Serverless Backend Architecture)

별도의 백엔드 가상 서버(Node.js/Python 호스팅)를 개설하고 운영 비용을 지출하는 대신, 모든 백엔드 기능을 Supabase의 서버리스(Serverless BaaS) 솔루션으로 통합 구현하여 인프라 비용을 최소화하고 개발 속도를 극대화합니다.

```
┌────────────────────────────────────────────────────────┐
│                      Supabase BaaS                     │
│  ┌────────────────┐ ┌────────────────┐ ┌────────────┐  │
│  │   PostgreSQL   │ │  GoTrue Auth   │ │EdgeFunction│  │
│  │ (퀴즈/프로필)  │ │ (구글/이메일)  │ │(영수증검증)│  │
│  └────────────────┘ └────────────────┘ └────────────┘  │
└───────────────────────────▲────────────────────────────┘
                            │ (HTTPS / Webhook)
                            │
                      ┌─────┴──────┐
                      │  ChartMon  │
                      │ (Android)  │
                      └────────────┘
```

1. **데이터베이스 & API (PostgreSQL)**:
   * `quizzes` 및 `profiles` 테이블을 저장하며, 클라이언트 SDK가 직접 보안 토큰(JWT)과 Row Level Security (RLS) 정책을 통해 DB를 조회하므로 별도의 REST API 컨트롤러 코드를 개발할 필요가 없습니다.
2. **회원가입 및 사용자 인증 (Supabase Auth)**:
   * 구글 소셜 로그인 및 이메일 가입 기능을 별도 보안 서버 없이 처리해 줍니다.
3. **구글 플레이 결제 검증 (Supabase Edge Functions)**:
   * 사용자가 구독 결제 시 구글로부터 획득한 영수증 토큰을 검증하기 위해 **Supabase Edge Function (TypeScript/Deno)**으로 가벼운 서버리스 검증 API를 배포합니다.
   * 이 에지 함수는 구글 API 자격 증명을 안전하게 관리하고 구글 인앱 결제 상태 검증 API를 대신 호출해 준 뒤 프로필의 구독 등급을 직접 승인 및 업데이트합니다.

## 5. 플레이스토어 업로드 규격 및 준비물 (Google Play Console Requirements)
* **개발자 프로필**: 법인 이름, 이메일, 전화번호, 법인 사업자등록증 사본.
* **스토어 등록 정보**:
  * **앱 이름**: ChartMon (차트몬)
  * **간단한 설명**: 하루 15분으로 차트 트레이딩 실력을 기르는 듀오링고형 훈련 앱.
  * **자세한 설명**: 주식 차트 분석, 아직도 책으로만 공부하시나요? 50가지가 넘는 실제 차트 사례 퀴즈, 출석 스트릭 보상, 상세한 전문 분석 해설까지! 리스크 없이 뇌에 트레이딩 감각을 이식하세요. ChartMon으로 시작하세요.
* **그래픽 자산**:
  * 고해상도 아이콘 (512x512 PNG, 알파 채널 없음).
  * 그래픽 이미지 (1024x500 JPG/PNG).
  * 휴대전화 스크린샷 (최소 2개 이상, 16:9 또는 9:16 비율).
* **빌드 아티팩트**: Android Studio에서 서명된 릴리스용 `.aab` (Android App Bundle).
