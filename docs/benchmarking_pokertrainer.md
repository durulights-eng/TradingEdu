# Benchmarking Report: PokerTrainer.se & ChartMon Redesign Proposal

본 문서는 유저가 제안한 벤치마킹 서비스인 **Poker Trainer (pokertrainer.se)**의 분석 결과와 이를 바탕으로 한 **ChartMon (차트몬)**의 아키텍처 및 UI/UX 전면 개편 제안을 다룹니다.

---

## 1. Poker Trainer (pokertrainer.se) 심층 분석

### 1) 서비스 개요
* **핵심 슬로건**: *"모던한 방식으로 포커를 연습하세요 (Practice poker in a modern way)"*
* **컨셉**: 포커를 단순 이론 책으로 배우는 대신, 특정 훈련 시나리오(Preflop, Postflop, Odds 등)를 반복 연습할 수 있는 **'시뮬레이션 체육관(Gym)'** 형태로 제공합니다.

```
[포커 트레이너 핵심 구조]
   ├─ Preflop Drill (오프닝/콜/3벳 범위 훈련)
   ├─ Postflop Simulator (핸드 리딩 및 에퀴티 연산)
   ├─ Hand Ranking Drill (핸드 세기 빠른 측정)
   ├─ Best Hand Drill (3개 패 중 위너 찾기)
   └─ Odds Drill (오즈 및 아웃츠 연산)
```

### 2) 성공적인 마케팅 및 게이미케이션 디자인 분석
* **시나리오 반복을 통한 근육 기억(Muscle Memory) 배양**:
  * 한 번 풀고 끝나는 퀴즈 북이 아니라, 랜덤으로 계속 주어지는 핸드 족보와 상황 속에서 즉각적인 결정을 내리게 합니다.
  * 유저는 자신의 결정이 정답(최적 플레이)인지에 대한 피드백을 실시간으로 확인하며 실력을 교정합니다.
* **대량의 문제/상황 풀**:
  * 단순 정적 데이터셋이 아니라, 딜러가 카드를 나눠주듯 다양한 상황이 동적으로 생성되거나 방대한 문제 은행을 가지고 유저가 질리지 않게 무제한 플레이를 지원합니다.
* **지속 가능한 지표 관리**:
  * 각 훈련별 레벨(Level), 최고 점수(High Score), 정확도(Accuracy), 스트릭(Streak)을 기록해 유저로 하여금 '기록 갱신'과 '실력 향상'의 직관적인 동기를 제공합니다.

## 2. 기존 ChartMon의 문제점 진단

* **선형 교과서식 커리큘럼의 한계**:
  * 현재 대시보드 하단에 나열된 6개의 정적 모듈은 **"한 번 풀면 더 이상 풀 이유가 없는 정적 시험지"**처럼 느껴집니다.
  * 트레이딩 역시 포커와 같이 빠르게 차트를 훑고 최적의 매수/매도 결정을 내려야 하는 **'직관과 근육 기억'의 영역**인데, 정적 교과서 구조는 리텐션(재방문율)을 크게 저해합니다.
* **학습 풀의 협소함**:
  * MVP용 6개 퀴즈로는 일일 3문제 데일리 훈련을 2일만 돌려도 중복 문제가 발생합니다. 문제 데이터와 시나리오가 **훠어얼씬 방대**해야 합니다.

## 3. ChartMon의 핵심 개편 제안: "차트 트레이닝 체육관 (Chart Training Gym)"

기존의 6개 커리큘럼 리스트를 과감히 제거하고, PokerTrainer처럼 **"카테고리별 무한 훈련 드릴(Drills)"**을 전면에 내세웁니다.

```
┌────────────────────────────────────────────────────────┐
│                        ChartMon                        │
│                                                        │
│   [ 🔥 연속 스트릭 ]     [ 🏆 내 경험치 ]   [ 👑 등급 ]  │
│                                                        │
│ ┌────────────────────────────────────────────────────┐ │
│ │ ⚡ 오늘의 맞춤형 데일리 워크아웃 (3문제)           │ │
│ └────────────────────────────────────────────────────┘ │
│                                                        │
│   ■ 실전 훈련 체육관 (Choose Your Drill)               │
│ ┌──────────────────────┐    ┌──────────────────────┐   │
│ │ 🌱 캔들 패턴 인지    │    │ 📈 지지/저항 & 돌파  │   │
│ │  Lv.3 | 최고 86%     │    │  Lv.1 | 최고 60%     │   │
│ └──────────────────────┘    └──────────────────────┘   │
│ ┌──────────────────────┐    ┌──────────────────────┐   │
│ │ 🔍 추세선 & 채널     │    │ 🛡️ 차트 패턴 포착    │   │
│ │  잠금 해제 대기      │    │  잠금 해제 대기      │   │
│ └──────────────────────┘    └──────────────────────┘   │
│ ┌──────────────────────┐    ┌──────────────────────┐   │
│ │ 👑 보조지표/다이버   │    │ ⚖️ 자금관리/손익비   │   │
│ │  잠금 해제 대기      │    │  잠금 해제 대기      │   │
│ └──────────────────────┘    └──────────────────────┘   │
│                                                        │
└────────────────────────────────────────────────────────┘
```

### 1) 6대 실전 훈련 드릴 (Chart Drills) 정의

1. **캔들 패턴 인지 드릴 (Candlestick Pattern Drill)**:
   * **목표**: 차트에 나타난 마지막 캔들이 어떤 형태(망치형, 도지, 상승장악 등)이고 다음 반전 가능성이 높은지 3초 안에 빠르게 판단.
2. **S&R 및 돌파 판단 드릴 (S&R & Breakout Drill)**:
   * **목표**: 차트 속 돌파 시도 지점을 보고 이것이 진짜 돌파(Breakout)인지, 개미를 낚는 가짜 돌파(Bull/Bear Trap)인지 판별.
3. **추세 및 채널 작도 드릴 (Trendline & Channel Drill)**:
   * **목표**: 지지/저항 채널 안에서 파동이 움직일 때 현재 채널의 상/하단 타점인지, 추세 이탈로 봐야 하는지 판단.
4. **차트 패턴 포착 드릴 (Chart Pattern Drill)**:
   * **목표**: 형성 중인 기하학적 패턴(헤드앤숄더, 불플래그, 이중 바닥 등)을 인지하고 넥라인 이탈 시점의 매매 뱡향성 결정.
5. **보조지표 다이버전스 드릴 (Indicator & Divergence Drill)**:
   * **목표**: 캔들 움직임과 아래 RSI/MACD 보조지표의 저점/고점 불일치를 보고 상승/하락 다이버전스를 빠르게 포착.
6. **자금관리 및 손익비 드릴 (Risk & Sizing Drill)**:
   * **목표**: 주어진 계좌 잔고, 손절폭(-N%) 정보를 보고 2% 룰 기반의 올바른 베팅(진입) 금액 및 최적의 손익비 타점을 신속 계산.

### 2) 핵심 UI/UX 플로우 개편
* **이론 공부의 분리**:
  * 대시보드 첫 화면에서 글을 읽는 탭을 지우고, 대신 훈련 드릴 내부 상단에 `[💡 이론 사전 보기]` 버튼을 팝업 형태로 넣어, 훈련하다가 헷갈릴 때 즉시 참고할 수 있게 서포트합니다.
  * 별도의 **"이론 백과 (Theory Guide)"** 탭을 하단 네비게이션에 추가하여 백과사전식 조회를 지원합니다.
* **드릴별 레벨 및 진행도(Progression)**:
  * 각 드릴은 10문제 단위의 세션으로 진행되며, 세션 클리어 시의 정확도에 따라 해당 드릴의 레벨과 최고 스트릭이 올라갑니다.

## 4. 데이터베이스 및 콘텐츠의 방대화 계획

현재 6개의 퀴즈 풀을 Supabase 및 로컬 상에서 **최소 60개 ~ 100개 이상**으로 대폭 확장합니다.

* **카테고리별 10~15개 이상의 차트 데이터셋 구축**:
  * [docs/gpt_quiz_generation_guide.md](file:///d:/TradingEdu/docs/gpt_quiz_generation_guide.md)의 가이드를 고도화하여 역사적 실제 차트(비트코인 숏스퀴즈, 테슬라 갭상승, 애플 지지선 리테스트 등)의 캔들 데이터를 JSON 좌표 형태로 대량 추가 주입합니다.
* **데이터셋 템플릿 표준화**:
  * 문제 은행에 다양한 난이도(상/중/하) 플래그를 추가하여 유저의 실력 레벨(Tiers)에 맞춰 동적으로 매칭되도록 쿼리를 구현합니다.

## 5. 실행 로드맵 (Execution Roadmap)

* **Phase 1**: UI 프레임워크 개편 (대시보드에 6대 드릴 카드 배치, 개별 드릴 진입/진행 로직 구현, 이론 읽기를 드릴 안 또는 백과사전 탭으로 이관).
* **Phase 2**: 문제 데이터 대량 확충 (Supabase DB에 약 50~60개 실전 퀴즈 추가 업로드).
* **Phase 3**: 난이도 세그멘테이션 및 드릴 전용 게이지/통계판 구현.
