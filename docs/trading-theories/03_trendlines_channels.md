# 03. 추세선과 평행 채널 작도 이론 (Trendlines & Channels)

트레이딩 명언 중 **"추세는 당신의 친구다 (Trend is your friend)"**라는 말이 있듯이, 시장의 현재 흐름이 상방, 하방, 혹은 횡보 중인지를 명확히 식별하는 것이 기술적 분석의 기본입니다. 추세선과 평행 채널은 고점과 저점의 규칙적 정렬을 통해 추세를 가시화합니다.

---

## 1. 올바른 추세선 작도법과 규칙 (Trendline Drawing)

추세선(Trendline)은 주가 움직임의 주요 저점들을 잇거나(상승 추세선), 고점들을 이어(하락 추세선) 현재 시장의 지배적인 힘의 방향을 나타내는 사선형 지지/저항선입니다.

```
 [ 상승 추세선 (Uptrend Line) ]          [ 하락 추세선 (Downtrend Line) ]
              고가                               고가  \
             /    \                                    \  고가
     저점   /      \  저점                              \   /  \  고가
    o------o--------o  <- 동적 지지선                    o---o---o  <- 동적 저항선
```

* **상승 추세선 (Uptrend Line)**: 점점 높아지는 저점(Higher Lows)들을 연결하여 그린 선입니다. 이 선은 가격 하락 시 **동적 지지선(Dynamic Support)** 역할을 합니다.
* **하락 추세선 (Downtrend Line)**: 점점 낮아지는 고점(Lower Highs)들을 연결하여 그린 선입니다. 이 선은 가격 상승 시 **동적 저항선(Dynamic Resistance)** 역할을 합니다.

### 추세선의 신뢰성을 보장하는 4대 철칙
1. **3점 테스트 법칙 (3rd Touch Rule)**: 임의의 두 점만 있으면 직선을 그을 수 있으므로 2개의 점만 연결한 사선은 단지 '가설'에 불과합니다. 반드시 주가가 되돌아와 **세 번째 터치(3rd Touch)에서 반등 혹은 저항에 성공**해야 비로소 시장 전체가 신뢰하는 유효한 추세선으로 승격됩니다.
2. **꼬리 vs 몸통 일관성**: 추세선을 작도할 때 캔들의 꼬리 끝을 이을지, 혹은 종가 몸통 끝을 이을지 일관되어야 합니다. 일반적으로 장중의 극단적 가격 합의 한계점까지 포함하는 **꼬리 끝(Wick to Wick)** 연결 방식이 가장 널리 받아들여지며 신뢰도가 높습니다.
3. **각도의 지속 가능성**: 추세선의 경사가 너무 가파르면(예: 45도 이상) 지속 시간이 매우 짧고 급격하게 붕괴될 확률이 큽니다. 가장 완만하고 지속력 높은 건강한 추세선은 대략 **20도~40도** 사이의 적절한 각도를 이룹니다.
4. **선형 스케일 vs 로그 스케일**: 가격 변동폭이 수십 배씩 발생하는 비트코인이나 급성장 기술주 차트의 경우, 일반 선형(Linear) 차트 대신 **로그(Logarithmic) 차트** 스케일을 켜고 추세선을 그려야 왜곡 현상 없이 먼 과거와 현재의 추세 지지/저항 라인을 정확하게 맞출 수 있습니다.

## 2. 평행 채널의 구조와 매매 기법 (Parallel Channels)

평행 채널(Parallel Channel)은 유효한 추세선과 평행한 사선을 반대편 고점/저점에 대칭 투사하여 만든 일종의 **기울어진 평행 통로**입니다.

```
┌──────────────────────────────────────────────┐  <- 채널 상단 (저항선: 매도/숏 타점)
│                    ▲                         │
│       ▲           / \           ▲            │
│      / \         /   \         / \           │
│  - - - - - - - - - - - - - - - - - - - - - - │  <- 채널 중심선 (Mid-line: 힘의 분곡점)
│    /     \     /       \     /     \         │
│   /       ▼   /         ▼   /       ▼        │
└──────────────────────────────────────────────┘  <- 채널 하단 (지지선: 매수/롱 타점)
```

* **상승 평행 채널 (Ascending Channel)**: 상승 추세선(지지선)을 복사해 고점들에 맞춘 평행선(저항선)을 씌운 구조입니다.
* **하락 평행 채널 (Descending Channel)**: 하락 추세선(저항선)을 복사해 저점들에 맞춘 평행선(지지선)을 씌운 구조입니다.

### 평행 채널 트레이딩 핵심 규칙
1. **채널 하단 (지지선)**: 강세 추세 속에서 가장 매력적인 **롱(매수) 진입 타점**입니다. 손절선은 채널 하단 이탈 직후로 아주 짧게 잡을 수 있어 손익비가 극대화됩니다.
2. **채널 상단 (저항선)**: 롱 포지션 보유자들의 주요 **분물 익절 구간**이며, 하락 채널이나 횡보 채널일 경우 짧은 스캘핑 관점의 **숏(매도) 포지션 진입 타점**으로 활용됩니다.
3. **채널 중심선 (Mid-line / 50% line)의 중요성**:
   * 중심선은 단기적인 지지와 저항선 역할을 수행합니다.
   * 주가가 채널 중심선을 상향 돌파하면 에너지가 상방 경계선까지 닿으려는 성질이 있으며, 반대로 중심선을 넘지 못하고 꺾이면 하단선을 깨뜨리러 내려갈 위험이 커집니다. 따라서 중심선은 **추세 강도의 가속/감속 판별 기준**입니다.

## 3. 추세선 이탈과 붕괴 (Breakout & Breakdown)

기존 추세선과 채널의 붕괴는 매우 급격한 시세 변동과 새로운 장기 추세의 탄생을 동반합니다.

* **추세선 하방 이탈 (Breakdown)**: 우상향 상승 추세선을 캔들 종가 기준으로 하방으로 뚫고 내려앉는 현상입니다. 이는 매수 주도권이 완전히 소멸하고 공급 폭탄이 시장을 잠식했음을 의미하므로 즉각적인 포지션 축소나 손절이 요구됩니다.
* **추세선 상향 돌파 (Breakout)**: 하락 추세 저항선을 장대양봉 종가 및 대량 거래량으로 상방 돌파하는 현상입니다. 기나긴 하락 관성을 이겨내고 새로운 상승 랠리로 회귀함을 뜻하므로 강력한 매수 신호가 됩니다.
* **추세 휩소 대비**: 사선 이탈 역시 수평선 돌파와 마찬가지로 가짜 이탈(Trap)이 빈발합니다. 따라서 **돌파된 추세선으로 다시 주가가 떨어져 리테스트 지지를 확인**하는 시점에 진입하는 것이 보수적인 트레이더의 최선책입니다.
