# ChartMon 작업 요약 (2026.06.30)

## Overview
이 문서는 차트몬(ChartMon / TradingEdu) 프로젝트에서 진행된 **데이터베이스(Mock Data) 및 애플리케이션 UI 로직 수정 사항**을 다른 코딩 에이전트(Antigravity, Codex 등)가 완벽하게 파악하고 후속 작업을 이어나갈 수 있도록 작성된 상세 인수인계서입니다.

## 1. Database (Quiz Data) Modifications
앱 내 문제 출제를 담당하는 정적 데이터(DB)에 대한 주요 수정 사항입니다.

- **Target File**: `src/data/quizzes/03_market_structure.ts`
  - **수정 사항**: 문제 및 해설 텍스트 내의 하드코딩된 가격(예: 150달러)을 대표님의 기획 의도에 맞게 **"3,000달러"** 등으로 전면 교체하여 현실성을 높였습니다.

- **Target File**: `src/data/quizzes/06_technical_indicators.ts`
  - **수정 사항**: 이동평균선(MA) 20일선과 50일선의 '골든크로스'를 묻는 문제에서, 캔들 3개만 표시되던 UX 오류를 수정했습니다.
  - **기술적 세부사항**: 토큰 낭비를 막기 위해 별도의 Node.js 스크립트(`inject_ma.cjs`)를 작성하여, 수학적으로 계산된 **20개의 캔들 배열**과 **20MA(노란색), 50MA(초록색) 곡선(Drawings) 궤적**을 자동으로 JSON 배열에 주입했습니다. 이제 차트 시각화 엔진이 이평선 궤적을 정상적으로 렌더링합니다.

## 2. App UI/UX & Component Modifications
React 컴포넌트 레벨에서의 UI/UX 디테일 및 렌더링 로직 수정 사항입니다.

- **Target File**: `src/components/ChartVisualizer.tsx`
  - **수정 사항**: 앱 전체의 차트 색상 테마를 **한국식(상승=빨강, 하락=파랑)**으로 통일했습니다.
  - **기술적 세부사항**: 
    - 상승(Bullish) 캔들 및 추세선: Red (`#ff5b67`)
    - 하락(Bearish) 캔들 및 추세선: Blue (`#3b82f6`)
    - 지지/저항(Support/Resistance) 추세선: Yellow (`#fbbf24`)

- **Target File**: `src/components/WelcomeOnboarding.tsx`
  - **수정 사항**: 앱 첫 실행 시 나타나는 온보딩 튜토리얼 화면의 디테일을 대폭 개선했습니다.
  - **기술적 세부사항**:
    1. 튜토리얼 내부에 하드코딩된 '함정 돌파' 모형 차트의 색상을 한국식(상승 빨강, 하락 파랑)으로 변경했습니다.
    2. **결제 페이지 시각적 계층화(Visual Hierarchy)**: 멤버십 가격 안내 텍스트를 수정하여, 메인 가격인 `월 14,900원`을 강조하고 부가 설명인 `(3개월 패키지 시...)` 부분을 `flex-column`을 활용해 아래 줄로 내리고 폰트 크기와 투명도를 조절했습니다 (구매 전환율 최적화).
    3. **가독성 개선**: PRO 멤버십 설명 텍스트 중 `패스권과` 뒤에 `<br />` 태그를 삽입하여 문장이 시각적으로 편안하게 끊어지도록 조치했습니다.

## 3. Build & Deployment Environment
다른 에이전트가 테스트 및 배포를 진행할 때 참고해야 할 명령어 파이프라인입니다.

- **Framework**: Capacitor + React (Vite)
- **Deployment Pipeline**:
  - 앱 내 UI 또는 데이터 변경 후에는 반드시 아래 명령어를 순차적으로 실행해야 에뮬레이터에 반영됩니다.
  ```bash
  npm run android:sync
  export JAVA_HOME="/opt/homebrew/opt/openjdk@21/libexec/openjdk.jdk/Contents/Home"
  cd android && ./gradlew assembleDebug
  adb -s [emulator_name] install -r app/build/outputs/apk/debug/app-debug.apk
  adb -s [emulator_name] shell monkey -p com.chartmon.app -c android.intent.category.LAUNCHER 1
  ```
- **State Management**: 온보딩 튜토리얼은 `localStorage`의 `chartmon_onboarding_completed` 값을 참조합니다. 튜토리얼 화면을 다시 테스트하려면 에뮬레이터에서 앱 데이터를 초기화해야 합니다 (`adb shell pm clear com.chartmon.app`).
